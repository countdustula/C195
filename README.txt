To run the program, first login with an INTEGER ONLY User ID and a password of any characters.  Once you've made it to the main screen, you are free add clients, add appointments,
update client or appointment information, delete appointments, and view reports.



•  a description of the additional report of your choice you ran in part A3f


I created a report that calculates the percentage appointments in each category.  On the left side of the screen, you can see what month has the most appointments, and which has the least.
On the right side of the screen, you can see if De-Briefings or Planning sessions are more common, as it gives you the percentage of what appointments are in each category.
